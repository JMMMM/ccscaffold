# 加载上一次会话

请执行以下步骤来加载上一次会话的内容：

1. **读取会话总结文件**
   读取 `.claude/conversations/session_summary.txt` 文件，获取上一次会话的总结。

2. **读取会话内容文件**
   读取 `.claude/conversations/conversation.txt` 文件，获取上一次会话的详细内容。

3. **读取文件修改记录**
   读取 `logs/modify_logs.txt` 文件，获取上一次会话的文件修改记录。

4. **整合并展示**
   将以上内容整合后，用中文向用户展示上一次会话的主要信息，包括：
   - 会话时间和主题
   - 主要讨论内容
   - 完成的任务
   - 文件修改情况
   - 遗留问题或后续计划

5. **询问用户**
   询问用户是否要基于上一次会话的内容继续对话，或者开始新的会话。
